# El Internet Antes de la Web
En este repositorio se realizaron varios ejercicios relacionados a cómo funcionaba
la web antes del internet. Principalmente se utilizó el sitio de telehack.com para
probar los inicios y cómo funcionaba antes la web.

Los ejercicios fueron resueltos totalmente (quitando el primero [^1]) dentro
de la terminal de wezterm. Las respuestas a los problemas resueltos están en el
pdf en `/docx/`, en el cual también se encuentran las capturas de los distintos 
outputs de cada problema.

[^1] El primer ejercicio lo resolví utilizando tanto el intérprete de brainf*ck
de telehack.com como mi propio intérprete en `/src/`. Mi intérprete no está completo
para cualquier programa de brainf*ck, pues no implementé los conteos necesarios para
juntar cada par de [] en los casos de loops debido a que el código requerido únicamente
tenía un set de [].
